3) SELECT et DISTINCT

  • Pour avoir le contenu d'une table SELECT * FROM employes;
  • Sélèctionner les prénoms et les noms des employés SELECT prenom, nom FROM employes;
  • Afficher les données sans les doublons avec le mot-clef DISTINCT SELECT DISTINCT(Service) FROM employes;

Les requêtes ne sont pas sensibles à la casse mais une convention indique qu'il ne faut mettre les mots-clefs des requêtes en majuscules. C'est plus facile ainsi de s'y retrouver.

4) Les conditions

a) Le mot-clef : WHERE

  • Nous cherchons une condition. Exemple : les prénoms et les noms des employés du services informatique : SELECT prenom, nom FROM employes WHERE service ='informatique';
  • Nous cherchons les employés du sexe Masculin : SELECT prenom, nom FROM employes WHERE sexe = 'm';
  • Nous trouvons la personne qui gagne 1390 euros : SELECT prenom, nom, service FROM employes WHERE salaire = '1390';

b) Les mots-clefs : BETWEEN et AND

  • Nous cherchons une condition une partie des données ex : les employés embauchés entre 2006 et 2010 : SELECT prenom, nom, date_embauche FROM employes WHERE date_embauche BETWEEN ='2006-01-01' AND '2010-12-31';
  • Dans une BDD les dates sont toujours notées au format USA YYYY-MM-DD

c) Le mots-clef : LIKE

  • Afficher les emploés dont le prénom commence par S : SELECT prenom, nom FROM employes WHERE prenom LIKE 's%';
  • Afficher les employés qui ont été embauché en 2000 SELECT * FROM emplyes WHERE date_embauche LIKE '%2000%';

Avec la présence du pourcentage avant et après le caractère recherché (ou les caractères recherchés), nous demandons au système de trouver des enregistrements contenant l'expression recherchée.

d) Le mots-clef : CURDATE()

  • Une requête qui done la date du jour SELECT CURDATE();
  • Les embauchés entre 2010 et aujourd'hui SELECT * FROM employes WHERE date_embauche BETWEEN '2010-01-01' AND CURDATE();

e) Le mots-clef : IN() avec plusieurs valeurs

  • Trouver les employés de la comptabilité et du service informatique : SELECT prenom, nom, service FROM employes WHERE service IN('informatique','comptabilite');
  • = (égal) permet d'annoncer une seule valeur tandis que IN permet d'annoncer plusieurs valeurs
  • trouver les personnes qui ne sont pas au service comptabiité et informatique : SELECT prenom, nom, service FROM employes WHERE service NOT IN('informatique', 'comptabilite');

f) Le mots-clef : HAVING

  • Ici on veut sélectionner seulement les services qui ont plus de 2 employés : SELECT service, COUNT(*) AS nombre_employes FROM employes GROUP BY service HAVING COUNT(*) >= 2;

6) Classement par ORDER BY

  • Grâce à ORDER BY on va classer les données par ordre alphabétique.
  • On range les noms par ordre alphabétique grâce au code suivant : SELECT prenom, nom FROM employes ORDER BY nom DESC,
  • On précise ASC ou DESC. Si ce n'est pas précisé ce sera trié autoùatiquement du plus petit au plus grand.
  • Fonctionne pour les chaîne de caractères comme pour les nombres : SELECT sexe, salaire FROM employes ORDER BY salaire ASC;
  • ORDER BY permet aussi de trier sur plusieurs colonnes : SELECT nom, prenom, service, salaire FROM employes ORDER BY service ASC, salaire ASC;

7) Limiter les résultats avec LIMIT

  • Afficher les employés en les classant par salaire (du + grand au + petit) et en limitant le résultat à 3 : SELECT nom, prenom, service, salaire FROM employes ORDER BY salaire DESC LIMIT 0,3;
  • Le premier chiffre précise l'enregistrement de départ : 0 définit le premier enregistrement informatique. Le sercond chiffre nous indique combien de résultats nous attendons.

8) Faire des calculs

  • Afficher la liste des employés avec leur salaire annuel : SELECT nom, prenom, salaire, salaire*12, FROM employes ORDER BY salaire DESC;

Utiliser un alias : AS

  • Avec le code précédent on a une colonne provisoire qui n'a pas de nom défini
  • Grâce au mot clef AS, on créé une colonne ponctuelle : SELECT nom, prenom, salaire, salaire*12 AS salaire_annuel FROM employes;

Calculer une somme SUM et une moyenne AVG

  • On calcul la somme totale des salaire à l'année des employés : SELECT SUM(salaire*12) FROM employes;
  • La même requete avec AVG : SELECT AVG(salaire) FROM employes;

Arrondir avec ROUND()

  • Avec ROUND on peut arrondir le résultat d'un calcul SELECT ROUND(AVG(salaire)) AS salaire_moyen_mensuelle FROM employes;
  • On peut cumuler deux notions avec ROUND et AVG; Attention à la syntaxe et aux parenthèses

Compter avec COUNT()

  • On peut compter le nombre d'emploées du sexe féminin : SELECT COUNT(*) FROM employes WHERE sexe = 'f';

Isoler une valeur MIN ou MAX

  • Avec MIN on peut trouver le salaire le plus bas : SELECT MIN(salaire) FROM employes;
  • Avec MAX on peut trouver le salaire le plus élevé :SELECT MAX(salaire) FROM employes;
  • Le système va exécuter en premier la requête entre parenthèse pour connaittre le résultat et l'exécuter comme source pour la seconde requete

9) Les codnitions multiples

  • Nous chercons les emploés SELECT nom, prenom, salaire FROM employes WHERE service = 'commercial' AND salaire >2000;
  • Lorsqu'il y a AND et OR au sein de la même requête, le système lit la condition de la droite vers la gauche :SELECT nom, prenom, service, salaire FROM employes WHERE service = 'production' AND salaire = 1900 OR salaire = 2300;
  • Pour garantir l'ordre naturel de lecture (de la gauche vers la droite), on utilisera les parentheses; SELECT nom, prenom, service, salaire FROM employes WHERE service = 'production' AND (salaire = 1900 OR salaire = 2300);

10) Les regroupements avec GROUP BY

  • Ici on veut connaitre le nombre d'emploés par service : Slect service, COUNT(*) AS nombre_employes FROM employes GROUP BY service ORDER BY nombre_employes;
  • La virgule signifie "et" donc on cherche le service et on compte le nombre d'emploés par service
  • Lorsqu'il y a AND et OR au sein de la même requête, le système lit la condition de la droite vers la gauche :SELECT nom, prenom, service, salaire FROM employes WHERE service = 'production' AND salaire = 1900 OR salaire = 2300;
  • Pour garantir l'ordre naturel de lecture (de la gauche vers la droite), on utilisera les parentheses; SELECT nom, prenom, service, salaire FROM employes WHERE service = 'production' AND (salaire = 1900 OR salaire = 2300);
