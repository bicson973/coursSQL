1/ MySQL

MySQL peut s'utiliser en ligne de commande avec :

  • Shell
  • Bash
  • CMDR
  • Un terminal Mac ou Linux
  • Ou une interface graphique comme phpMyAdmin

Pour ce connecter en ligne de commande on utilise :
mysql -u utilisateur -p

Attention :
il faut penser à mettre le nom d'utilisateur spécifique au serveur ; sur le pc en local l'utilisateur est "root" qui signifie "racine".

En local sur pc, par défaut il n'y a pas de mot de passe, sur Mac le mot est 'root'

En ligne chaque BBD aura son utilisateur propre et son mot de passe propre
